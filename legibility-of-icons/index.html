Legibility of Icons


Concept Design, 2024.

Icons, the abstraction of objects. Information was compressed to make an icon. We see them as a representation of the physical things. It contains so much information in such simple forms, usually lines and geometric shapes. But to an extent the compression of information starts to make them illegible. The project aims to explore this ambiguous boundary.

Rule
1. Abstract shapes from the physical object.
2. Place them into the coordinate system.
   a. Straight lines
      i. Parallel to the x / y / z axis: Give the color of the axis to the line
      ii. Not parallel to any: NG
   b. Curves: Use the coordinate plane to cut the shape. The cut curves are given the color of the axis which is closer as a parallel.
3. Reduce one set of information, leaving two colors.
e.g. Reduce the x axis -> G & B
4. Reduce another set of information, leaving one color.
e.g. (Reduce the x axis.) Reduce the y axis -> B
5. Decide if Legible / Not legible
